A cozy, minimal top-down farming game made with Love2D.

## How to Play

### Controls
- **1** - Select Hoe
- **2** - Select Seeds
- **3** - Select Watering Can
- **4** - Select Hand
- **Left click** - Use current tool

## Running the Game

Make sure you have [Love2D](https://love2d.org/) installed, then:

```bash
love .
```
